## 신발데이터 라벨링 프로젝트
### 사용한 스택
1. Programing Languages
    + Python
    + HTML
    + CSS
    + JS
2. Framework/Library
    + Django
    + jQuery
    + Bootstrap
3. Server
    + vultr
    + MariaDB
    + NginX
4. Tooling
    + docker
5. ETC
    + Notion
    + Figma
---
### 주요 기능
+ 작업자명 작성 후 작업 가능하도록 설정
+ 작업창에서 jquery를 사용하여 상위카테고리에 맞는 하위카테고리로 변경
+ 작업현황판
  + 작업한 데이터 현황 확인
  + csv, exel 두가지 버전 export
  + export 할 수 있는 데이터 미리보기
